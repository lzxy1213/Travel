# Travel

#### 项目介绍
一个旅游app，首页上实现了多区域轮播的功能以及多区域列表的循环展示，进入到城市展示页面,实现了城市的展示和搜索,以及城市右侧字母与左侧区块动画连动的效果,进入景点详情页面,实现了公用的画廊组件,渐隐渐现的header组件,以及递归展示的列表组件.
